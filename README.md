# Iskra_space

Живой шаблон на базе Very Good Core, настроенный под Искру.

## Быстрый старт
```bash
flutter pub get
flutter run -d chrome # или нужную платформу
```

## Workspace (Melos)
Пакеты Искры живут в `packages/*`. Команды:
```bash
dart pub global activate melos
melos bootstrap
melos run analyze
melos run test
```
